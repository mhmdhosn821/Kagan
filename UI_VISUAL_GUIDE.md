# راهنمای رابط کاربری جدید Kagan ERP

## 📸 نمای کلی تغییرات UI

این مستند شامل توضیحات بصری از تغییرات رابط کاربری است.

---

## 1️⃣ صفحه ورود (Login Page)

### ظاهر قبل از تغییر:
- پس‌زمینه خاکستری ساده (#ecf0f1)
- فریم سفید ساده
- فونت‌های کوچک (12px)
- دکمه آبی ساده

### ظاهر بعد از تغییر:
```

│   ║  رمز عبور (12px bold)             ║   │
│   ║  [                           ]    ║   │
│   ║  (50px height, glass effect)      ║   │
│   ╚═══════════════════════════════════╝   │
```

**ویژگی‌ها:**

- افکت blur روی پس‌زمینه
- دکمه ورود: گرادیانت آبی (#4facfe → #00f2fe)

## ✨ افکت‌های ویژه

### Glass Morphism
```css
background: rgba(255, 255, 255, 0.15);
backdrop-filter: blur(20px);
border: 1px solid rgba(255, 255, 255, 0.2);
border-radius: 16px;
box-shadow: 0 8px 32px rgba(0, 0, 0, 0.1);
```

### Gradient Buttons
```css
background: qlineargradient(x1:0, y1:0, x2:1, y2:0,
    stop:0 #4facfe, stop:1 #00f2fe);
```

### Hover Effects
- تغییر رنگ به تیره‌تر
- افزایش سایه
- تغییر cursor به pointer
